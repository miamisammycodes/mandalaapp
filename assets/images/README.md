## Image Guidelines

- **Format:** PNG or JPG
- **Size:** Optimize for mobile (each image should be < 500KB if possible)
- **Resolution:** Use 2x or 3x resolution for crisp display on all devices
- **Naming:** Use lowercase with underscores (snake_case)

## Notes

- Content images are optional - the app will show placeholders if they're missing
